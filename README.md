# React TodoList（使用 Redux Toolkit）

- 使用 Redux Toolkit 實作 TodoList，包含新增任務、刪除任務、狀態切換。

- [線上Demo](https://satarkuo.github.io/react-reduxToolkit-todolist/)

## 🚀 快速啟動

```bash
# 安裝套件
npm install

# 啟動專案
npm run dev
```

---

## 📁 專案資料夾結構簡介

- 本專案採用模組化設計，資料夾說明如下：
```txt
src/ 
├─ assets/      # 靜態資源（圖檔、樣式等） 
├─ component/   # 可重複使用的元件 
├─ slice/       # Redux Toolkit 狀態管理設定 
└─ store/       # Redux Toolkit 狀態管理設定 
```
